# COPD Longitudinal Analysis Project

## Overview
This repository contains the R code and statistical analysis workflow for a longitudinal cohort study focusing on Chronic Obstructive Pulmonary Disease (COPD).

## Research Objectives
- Investigate the trajectory of **hemodynamic parameters** (e.g., mPAP, PVR) over time.
- Perform survival analysis based on baseline clinical characteristics.

## Tools Used
- **Language:** R
- **Key Packages:** `lme4`, `survival`, `ggplot2`, `gtsummary`
